### Usage 

```
    git clone https://github.com/jasonz1987/laravel-flowbite-dcat-template.git your-app-new 
```

```shell
    composer install
```

```shell
  cp .env.examp .env
```

```shell
    php artisan key:generate
```

```shell
    npm install
```

```shell
   php artisan migrate
```

```shell
    npm run dev
```

if you want to generate your admin menu

```shell
    php artisan db:seed --class=MenuSeeder
```
